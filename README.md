# status-codes


## Install

```bash
npm install @woopidi/status-codes
```

## How to use

```node
import { OK, NotFound } from "@woopdi/status-codes";
```

## Codes

| Constant | Code |
| ------------ | ------------- |
| CONTINUE | 100 |
| SWITCH_PROTOCOLS | 101 |
| PROCESSING | 102 |
| EARLY_HINTS | 103 |
| OK | 200 |
| CREATED | 201 |
| ACCEPTED | 202 |
